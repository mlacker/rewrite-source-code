# rewrite-source-code

此项目用于对 Spring Framework, Spring Cloud, Java 并发等框架的研究和学习。

基于 Kotlin 语言良好的语意表达能力，实现相应框架核心功能的代码逻辑，并忽略一些与之无关的代码，以便于阅读和理解。

实践重于学习
