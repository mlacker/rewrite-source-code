# 分布式

## CAP

CAP 原则又称 CAP 定理，指的是在一个分布式系统中 Consistency（一致性), Availability（可用性）, Partition tolerance（分区容错性）三者不可兼得。

- 一致性（C）：在分布式系统中的所有数据备份，在同一时刻是否同样的值。
- 可用性（A）：在集群中一部分节点故障后，集群整体是否还能响应客户端的读写请求。
- 分区容忍性（P）：
